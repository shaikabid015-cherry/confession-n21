# Changelog

## [1.0.0] - 2024-01-20
### Added
- Complete confession website with all core features
- Multi-step confession form with emoji mood selection
- User authentication system
- Email notification system
- Dark/Light mode toggle
- Responsive design for all devices
- Messages management interface
- User statistics tracking
- Backend API server

### Features
- Send confessions anonymously or via ID
- Receive email notifications
- View received messages
- Track sent/received statistics
- Secure user sessions
- Modern UI with smooth animations

### Technical
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Node.js, Express, Nodemailer
- Storage: LocalStorage (frontend), JSON files (backend)
- Deployment: Vercel (frontend), Render (backend)
